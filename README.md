#### 5、让这成为一套开源的家庭OA系统

## 前期规划：
#### 1、一边学习，做到学以致用。

## 开发步骤：
### 1、先搭建主要的框架
### 2、搭建权限部分
### 3、用户的简单CRUD
### 4、原生JDBC+Servlet+JSP

## 技术选型：
### V1.0版本：MySQL+Servlet+JSP
### V2.0版本：MySQL+Spring+SpringMVC+JSP
